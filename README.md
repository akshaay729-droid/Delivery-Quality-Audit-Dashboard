# 🚚 Delivery Quality Audit Dashboard

## Overview

A Streamlit-based analytics dashboard for auditing delivery operations.

The dashboard identifies:

- Late Deliveries
- Failed Deliveries
- Address Issues
- Invalid Coordinates
- Driver Performance
- SLA Compliance
- Issue Severity Levels
- Root Cause Analysis

---

## Features

### Executive Summary
High-level delivery quality metrics.

### Quality Issues Overview
Visual breakdown of operational issues.

### Driver Performance Analysis
Driver-wise order distribution and scorecards.

### City Performance
City-wise order distribution.

### Issue Severity Analysis
Critical, Medium and Low severity issue tracking.

### Root Cause Analysis
Identification of operational bottlenecks.

### Orders Requiring Attention
Automatically flags problematic deliveries.

### Recommendations
Automated operational recommendations.

---

## Tech Stack

- Python
- Pandas
- Streamlit
- Matplotlib
- OpenPyXL

---

## Run Locally

```bash
pip install -r requirements.txt

streamlit run app.py
```
